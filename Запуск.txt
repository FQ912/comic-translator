# УСТАНОВКА И ЗАПУСК

## 1. Установка Python
- Скачайте Python 3.10+ с https://python.org
- При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"

## 2. Установка Tesseract OCR (для распознавания текста)
- Скачайте с https://github.com/UB-Mannheim/tesseract/releases
- Запустите установщик
- Отметьте "Additional language data" и "English"

## 3. Сборка проекта (установка зависимостей)

### Windows:
Откройте командную строку в папке с проектом и выполните:
```batch
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
